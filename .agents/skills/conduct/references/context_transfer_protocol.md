# Context Transfer Protocol

Make sessions interchangeable by transferring context through files, not memory.

Required files:

- `PROJECT_CONTEXT.md`: project purpose, architecture, commands, known traps.
- `TODO.md`: active, blocked, deferred, and completed tasks.
- `DECISIONS.md`: durable decisions and reasons.
- `LESSONS.md`: recurring mistakes and repo-specific invariants.
- `HANDOFF.md`: current status and next-session instructions.

A new session should continue by reading:

1. `HANDOFF.md`
2. `TODO.md`
3. `PROJECT_CONTEXT.md`
4. current task `FINAL.md` or `FAILURE_BRIEF.md`
