# Skill Preservation Protocol

When a model struggles, stop empty retries and preserve the lesson.

Use `create-skill-meta` to turn the lesson into:

- a new skill;
- a patch to an existing skill;
- a point-of-use checklist;
- a validation script;
- an AGENTS.md instruction;
- a project-specific `LESSONS.md` note.

Run this protocol when:

- same error repeats twice;
- same strategy is retried twice;
- model misses a repo convention;
- missing dependency blocks progress;
- task cannot finish within budget;
- reviewer finds recurring failure pattern.
