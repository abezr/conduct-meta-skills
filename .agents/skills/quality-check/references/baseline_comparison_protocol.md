# Baseline Comparison Protocol

A skill must improve outcomes enough to justify context and maintenance cost.

Run or simulate the same task twice:

1. Without the skill.
2. With the skill.

Compare:

- correctness;
- completeness;
- edge-case handling;
- validation behavior;
- tool usage;
- unnecessary clarification questions;
- output usability;
- token/context efficiency.

Pass rule:
- with-skill output is meaningfully better or safer;
- the skill does not over-trigger;
- the skill does not introduce stale or wrong guidance.
